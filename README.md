# ISO-TEI-Spoken: Tool Support

The project **Transcription+** (https://linguisticbits.de/transcription-plus.html) is revising, consolidating and publishing resources related to the ISO standard 
**ISO 24624:2016 Language resource management — Transcription of spoken language** (https://www.iso.org/standard/37338.html).

This repository gathers existing resources from different existing resources and makes them available as a library / API.
